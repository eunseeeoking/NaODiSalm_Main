# 나어디삶 — 공모전 기획서 (1차 초안)

> **부제**: 데이터 기반 청년·신혼부부 주거 의사결정 도구
> **출품**: 2026년 국토교통부 공공데이터 활용 공모전
> **작성**: 2026-05-24 (D-6, 1차 초안)
> **상태**: 골격 + 핵심 표 작성 완료. [채울 곳] 표시된 부분은 사용자 직접 입력.

---

## 1. 서비스 개요

### 한 줄 정의
> **"내 직장·소득·인내심을 입력하면, 나에게 맞는 동네를 자동 추천하는 청년 주거 의사결정 도구"**

### 해결하려는 문제
- 청년·신혼부부가 **첫 자가 매매·전세 결정**에서 직면하는 정보 격차
- 직방·다방 같은 시세 검색 서비스는 있지만, **"내 통근·소득·안전 종합 점수"로 동네를 추천**하는 서비스는 부재
- 부동산 의사결정에서 가장 큰 불안: "내 소득에 적정한가? 통근 가능한가? 안전한가?"

### 핵심 가치 제안
1. **다축 점수 추천** — 통근(transit) + 부담(affordability/RIR) + 안전(safety) + 생활(life) 4축 가중합
2. **4기관 공공데이터 융합** — 국토부 RTMS + 한국부동산원 R-ONE + 한국교통안전공단 TAGO + LH 청년주택
3. **AI 가격 분석** — ARIMA 통계 모델로 단지별 3년 가격 안정성 정량화

### 타겟 사용자
- 1차: **사회초년생 + 신혼부부** (서울 직장 + 자가/전세 매매 검토)
- 2차: 자가 보유 직장인 (이사 검토)
- 3차: 정책 설계자 (청년 주거 정보 격차 분석)

## 2. 사회적 가치

### 청년 주거 위기 — 통계로 보는 현실

| 지표 | 수치 | 출처 |
|------|------|------|
| 서울 청년(19~39세) 1인가구 수 | **약 73만 가구** (서울 전체 가구의 13.5%) | 통계청 인구주택총조사 (2020), 서울 열린데이터광장 |
| 청년 임차가구 평균 RIR (소득 대비 주거비) | **17.4%** | 국토교통부·국토연구원 「2023 주거실태조사」 |
| 최저주거기준 미달 청년가구 비율 | **8.2%** (2024년 기준, 전년 대비 +2.1%p ↑) | 국토교통부 「2024 주거실태조사」 |
| 청년 1인가구 주거 불안 1위 요인 | **범죄·안전 (17.2%)** — 1인가구 최대 불안 요인 | 국가데이터처 「2024 통계로 보는 1인가구」 |

> **해석**: 서울 청년 임차가구 RIR 평균 17.4%는 전국 임차가구 평균(15.8%)보다 높음.
> 최저주거기준 미달 청년가구는 2024년 급증 추세 (+2.1%p)로 주거 위기 심화.
> 이들 73만 청년 가구 중 정보 격차로 적정 주거지 판단이 어려운 인구 = 본 서비스 핵심 타겟.

### 정보 격차의 본질
- 직방·다방·네이버부동산 → **개별 매물 시세** (어디가 좋은 동네인지 판단 X)
- 국토부 RTMS → **공공 거래 원본** (분석 도구 없음, 일반인 접근 어려움)
- 한국부동산원 R-ONE → **시군구 단위 통계** (단지 단위 추천 X)
- LH 청년주택 정보 → **별도 사이트**, 시세·통근 정보 연계 X

→ **"청년이 알아야 할 정보가 4개 기관에 분산"** = 본 서비스가 통합 제공

- 「공공데이터법」 (2013) §7 — 민간 활용 활성화

## 5. 기술 검증 — AI 모델 백테스트 ★

### 검증 설계
- **방법**: 시계열 hold-out (학습/평가 분리)
- **데이터**: RTMS 19년치 중 처음 196개월 학습 → 마지막 36개월 예측 → 실제값과 비교
- **단위**: 서울 거래량 상위 5단지 (다양한 자치구)
- **지표**: MAPE(평균 오차율) / RMSE(절대 오차) / R²(설명력)
- **horizon**: 36개월 (3년 예측)

### 비교 모델 3종
| 모델 | 위치 | 비고 |
|------|------|------|
| MA-12 | TypeScript | 12개월 이동평균 + 선형 외삽 (베이스라인) |
| ARIMA(2,1,2) | Python statsmodels | 통계학 표준 시계열 모델 |
| LSTM | TensorFlow.js | window=24 recursive multi-step |

### 결과 (서울 5단지 평균, horizon=36mo, train=196mo)

```
모델           MAPE    RMSE      R²       비고
─────────────────────────────────────────────
ARIMA(2,1,2)   10.16%  159.71   -1.75    ✅ 최강 — 학계 부동산 단기 예측 표준 수준
MA-12          10.88%  165.39  -10.90    🟡 단순 외삽이지만 안정
LSTM           20.41%  283.69  -204.02   🔴 시계열 변동성 학습 한계 노출
```

> **별첨**: `reports/plots/summary.png` (3-패널 막대그래프)

### 정직한 해석
1. **ARIMA가 부동산 단기 예측에 가장 적합** — 차분으로 추세 분리, 적은 파라미터로 안정 학습
2. **LSTM은 단순 데이터 양 증가로 해결 안 됨** — 2008 폭락 / 2020 폭등 / 2022 조정 등 국면 전환을 24개월 윈도우로 학습 어려움
3. **학계 보고 부동산 단기 MAPE**: ARIMA 8~15%, LSTM 7~18% (데이터·구조에 따라 변동 매우 큼)
   → 본 서비스 ARIMA 10.16%는 **학계 평균 수준**, 정직한 성능

### 향후 개선 방향 (논문성 강화)
- Multivariate LSTM (금리·정책·인구이동 외생 변수 결합)
- Regime-switching ARIMA (국면 인지 자동)
- Multi-complex transfer learning (단지간 패턴 전이)

### R-ONE 정규화 사례 보고
- 강북구 SK북한산시티 단지에서 LSTM + R-ONE 정규화 시 MAPE 9.41% → 8.54%, R² -0.51 → -0.24
- 시장 추세 약한 외곽 지역에서 정규화 효과 확인
- 송파·잠실 단지에서는 효과 X (시장 추세 자체가 강한 신호)
- → **"단지·지역 특성별 정규화 정책"** 후속 연구 방향

### 중기 (1~2년) — B2C + B2G 확장
- **B2C**: 프리미엄 분석 (광고 X, 구독제 — 월 5,000원 ~ 1만원)
- **B2G**: 지자체·LH 협력 (청년 주거 정책 의사결정 지원 도구)
- **API 개방**: 4축 점수를 다른 부동산 앱에 제공

### 장기 (3~5년) — 전국 확장
- 서울 25구 → 광역시 7개 → 전국 시·군 확장
- 부동산 외 영역 (전월세 매물 추천, 동네 라이프스타일 매칭)
