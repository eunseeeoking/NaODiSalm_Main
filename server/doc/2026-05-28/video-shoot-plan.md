## 4. GIF 인코딩 명령 (참고)

원본 MP4 → GIF 변환 (ffmpeg + palettegen):

```powershell
# 1) palette 생성
ffmpeg -i demo.mp4 -vf "fps=12,scale=960:-1:flags=lanczos,palettegen" -y palette.png

# 2) palette 적용 GIF 출력
ffmpeg -i demo.mp4 -i palette.png -filter_complex "fps=12,scale=960:-1:flags=lanczos[x];[x][1:v]paletteuse" -y demo.gif

# 3) gifski (선택, 더 작은 용량 + 더 좋은 색감)
gifski -o demo.gif --fps 12 --width 960 demo.mp4
```

목표 용량: 5~8MB. 초과 시 fps 12 → 10, 폭 960 → 800 단계 조정.

ScreenToGif 사용 시:
- 캡처: 25fps → GIF 12fps (스킵 1프레임)
- 색상: 256 → 128 (용량 ↓)
- 도구 → "최적화 → 색상 양자화 + 프레임 중복 제거"
