## 1. 왜 전환하는가

### 1.1 공공데이터 공모전 취지와의 거리

```
공모전 정체성:  국토교통부 / 공공데이터 / 사회적 가치 / 정책 시너지
기존 컨셉:      "직장인 + 투자 수익률" → 사적 자산 증식 도구

→ 사적 투자 도구는 공공데이터 활용의 정당성 약화
→ 평가위원 입장에서 "왜 굳이 공공데이터로 투자 도구를?" 의문 발생
→ 사회적 영향 / 공공성 항목에서 감점 불가피
```

### 1.2 시장 포지셔닝 한계

```
2024년 대상작 "바로(BALAW)"  ← 부동산 + AI 카테고리 이미 천장 점유
                              유사 컨셉으로 재도전 시 차별화 어려움

직방·다방·바로 등 민간 매물 서비스가 즉시 비교 대상으로 떠오름
→ 독창성 항목 14~17/30 (보수 평가)
```

### 1.3 평가 기준 재해석

| 항목 | 배점 | 기존 컨셉 추정 | 청년 타겟 추정 |
|---|---|---|---|
| 독창성 | 30 | 14~17 | 20~24 |
| 구체성 | 30 | 22~25 | 26~29 |
| 성장성 | 40 | 18~23 | 28~34 |
| 가점 | +15 | +10 | +15 |
| **합계** | **115** | **64~75** | **89~102** |

성장성의 "사회적 영향·기여도(10점)" + "창업·기술실현 가능성(20점)" 두 축이 결정적.
청년 주거 빈곤 / 정보 격차 해소 / 1인가구 안전망은 정책 시너지 직접 부합.

## 2. 새 정체성

### 2.1 슬로건 / 한 줄 소개

```
[기존]   "매물이 아니라 사람을 분석합니다."  (사람=직장인=투자자)
[신규]   "청년·신혼부부의 첫 집 찾기 — 데이터로 안전하게."

서비스명:   "나어디삶"  (유지 — 1인칭 질문 그대로)
부제:       "데이터 기반 청년 주거 의사결정 도구"
```

### 2.2 타겟 페르소나 (재정의)

```
[1] 사회초년생 (1인가구, 25~32세)
    · 첫 직장 + 첫 자취
    · 통근시간 + 주거비 부담 + 안전이 핵심 고민
    · LH 행복주택 / 청년매입임대 / 전세임대 후보 자동 탐색 필요

[2] 신혼부부 (맞벌이, 28~35세)
    · 두 직장 통근 균형점 탐색
    · 신혼부부 전세대출 / 디딤돌 한도 내 매물
    · 향후 출산 대비 학군·생활편의

[3] 직장인 (유지, 4번째 우선순위)
    · 기존 페르소나 유지하되 우선순위 낮춤
    · 핵심 기능은 청년·신혼 흐름 안에 포섭
```

### 2.3 핵심 가치 명제 (Value Proposition)

```
Before:  "통근 + 가성비 + 투자 + 생활편의 가중치 → 매물 추천"
After:   "통근 + 소득 대비 주거비 + 안전 + 생활편의 가중치 → 청년 주거 추천"

핵심 변화:
  · "투자(LSTM 3년 수익률)" 축 폐기
  · "안전(1인가구 범죄·가로등·CCTV)" 축 신설
  · "가성비" → "소득 대비 주거비 부담률(RIR)" 로 의미 강화
  · LH 청년주택 매칭이 추천 결과의 일부로 통합
```

## 3. 평가 항목별 점수 영향 (보수 추정)

### 3.1 독창성 (+6~7점)

```
✓ 매물 분석이 아닌 "조건 입력 → 시스템 추천" — 기존 차별점 유지
✓ 청년·신혼부부 타겟 정보 통합 도구는 시장 공백 (직방·바로 미커버)
✓ 소득 대비 주거비 자동 계산 + 청년주택 매칭 + 안전 점수 — 신규 조합
```

### 3.2 구체성 (+4점)

```
✓ 공공데이터 4개 기관 융합 (국토부 + 한국부동산원 + LH + 통계청·경찰청)
✓ LSTM 비교 검증 (MA-12, ARIMA vs LSTM) → 모델 선택 정당화
✓ 소득 분위별 RIR 계산식 명시 가능
```

### 3.3 성장성 (+10~11점)

```
✓ 사회적 영향: 청년 주거 빈곤 / 정보 격차 해소 — 정책 명분 분명
✓ 실용성: 정책 자금(디딤돌·버팀목·LH) 매칭은 사용자가 곧장 행동 가능
✓ 창업 가능성: 청년 정책 사업 + 지자체 협업 + B2G 모델 가능
```

### 3.4 가점 (+5점)

```
✓ 데이터 융합(주관기관) — 국토부 + 한국부동산원 두 기관 융합 확정 시 +5
✓ AI 활용 — LSTM 학습·분석 (+10, 이미 확보)
```
